# Spaceship Titanic 数据集说明

## 数据集概述

这是一个来自 Kaggle 的机器学习竞赛数据集，主题是太空飞船泰坦尼克号。数据集中包含了乘客的个人信息，目标是根据这些信息预测乘客是否被传输到另一个维度。

## 数据文件说明

### 训练集 (train.csv)
- **行数**: 约 8700 行（约三分之二的乘客）
- **用途**: 训练机器学习模型
- **包含**: 特征列 + 目标变量 `Transported`

### 测试集 (test.csv)
- **行数**: 约 4300 行（约三分之一的乘客）
- **用途**: 预测并提交结果
- **包含**: 仅特征列（没有 `Transported`）

## 字段详细说明

### 标识信息
| 字段 | 说明 |
|------|------|
| **PassengerId** | 乘客唯一标识符，格式：`gggg_pp` |
|  | - `gggg`: 乘客所在的旅行组编号 |
|  | - `pp`: 组内成员编号 |
|  | - 提示：同一组的人通常是家庭成员，但并非总是 |
| **Name** | 乘客的姓名（名 + 姓） |

### 个人信息
| 字段 | 说明 |
|------|------|
| **HomePlanet** | 出发行星（通常是永久居住地） |
| **Age** | 乘客年龄 |
| **VIP** | 是否购买了VIP服务 |

### 旅行信息
| 字段 | 说明 |
|------|------|
| **CryoSleep** | 是否选择低温休眠 |
|  | - 是：航行期间被限制在舱内 |
|  | - 否：可以在飞船内活动 |
| **Cabin** | 舱位编号，格式：`deck/num/side` |
|  | - `deck`: 甲板编号（A, B, C, ...） |
|  | - `num`: 房间号 |
|  | - `side`: 侧舷（P=Port左舷, S=Starboard右舷） |
| **Destination** | 目的地行星 |

### 消费信息
| 字段 | 说明 |
|------|------|
| **RoomService** | 客房服务消费金额 |
| **FoodCourt** | 美食广场消费金额 |
| **ShoppingMall** | 购物中心消费金额 |
| **Spa** | 水疗中心消费金额 |
| **VRDeck** | VR甲板消费金额 |

### 目标变量
| 字段 | 说明 |
|------|------|
| **Transported** | 是否被传输到另一个维度 |
|  | - 是：目标变量为 True |
|  | - 否：目标变量为 False |
